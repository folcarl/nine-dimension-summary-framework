# Methodology

The nine-dimension framework treats summary as meaning reconstruction, not compression.

A source exists in three time relations:

- the time in which it was produced;
- the time or world it represents;
- the later moment from which readers interpret it.

It also carries form: interview, report, lecture, commentary, documentary, article, or another medium. Form changes what can be trusted, what gets omitted, and how audiences feel they are using the material.

## The Nine Dimensions

1. Production time.
2. Represented time.
3. Reading or viewing time.
4. Surface subject.
5. Form and medium.
6. Deeper problem.
7. Perceived object.
8. Viewing position.
9. Effect of viewing.

The output should not mechanically display nine boxes. A good summary reads naturally while making clear that time, object, form, evidence, audience, limits, and implications have been handled.
