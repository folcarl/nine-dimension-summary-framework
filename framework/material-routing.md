# Material Routing

Use this reference to choose material type, execution mode, and extra requirements.

Default rule: when uncertain, use full mode.

| Material type | Default mode | Required emphasis |
| --- | --- | --- |
| Short low-density text | Lightweight | Core content, real problem, form/viewing risk, one concrete limitation, 3-5 scores. |
| Long interview / oral history | Full | Person and situation, memory reliability, editing choices, silence and omissions, facts needing verification. |
| High-density lecture | Full | Concepts, argument path, citation/evidence network, scope of claims, possible over-extension. |
| Commentary / analysis video-derived text | Full unless clearly short | Emotional framing, "seeing through the truth" posture, evidence chain, alternative explanations, audience risk. |
| Investment / economic / research material | Full | Core variables, transmission mechanisms, assumptions, indicators, risks, no investment advice. |
| Political / historical material | Full | Historical context, institutional structure, competing positions, ethical cost, evidence boundary. |
| Artistic / film / documentary material | Full for complex work; lightweight for brief notes | Form/content relation, aesthetic strategy, emotional effect, narrative or ethical limits. |
| Batch of multiple materials | Full per material | Summarize each independently before any synthesis. |

## Mode Choice

Use full mode if any of the following are true:

- material is long or dense;
- material involves politics, history, economics, investment, policy, research, or public events;
- material relies on charts, data, reports, or claims that may need checking;
- user asks for a complete, rigorous, critical, or structured summary;
- there are multiple files;
- you are unsure.

Use lightweight mode only when all are true:

- source is short;
- information density is low;
- stakes are low;
- claims are simple;
- user did not request depth.
