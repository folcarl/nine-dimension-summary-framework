# Authorized Media Policy

This skill is an optional local transcription adapter. It is not the main product and should not be presented as a downloader.

Process media only when the user provides:

- a local file they are allowed to process; or
- a URL for content they are authorized to download or transcribe under applicable law and platform terms.

Do not store or commit:

- downloaded media;
- cookies or browser auth data;
- generated transcripts containing private content;
- logs or job state;
- temporary files.

If authorization is unclear, ask the user to confirm they have rights to process the content.

When transcript cleanup is requested, only correct obvious recognition errors. Mark uncertain names, numbers, institutions, and claims instead of silently rewriting them.
