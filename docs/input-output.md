# Input And Output

## Supported Inputs

The main skill supports:

- direct text;
- transcript files;
- articles;
- interviews;
- lectures;
- podcasts;
- research notes;
- reports;
- user-provided video-derived text.

The default path does not require a media URL.

## Full Mode Output

Full mode normally includes:

- Core Content / Core Argument
- The Real Problem It Handles
- Form And Viewing Logic
- Evidence Chain And Reasoning
- Critique And Limits
- External Verification Boundary
- Implications For Judgment And Action
- Follow-Up Observation List
- Overall Evaluation And Scores

## Lightweight Mode Output

Lightweight mode normally includes:

- Core Content
- Real Problem
- Form / Viewing Risk
- Main Limitation
- Overall Evaluation And Scores

Use lightweight mode only for short, low-density, low-stakes material.
