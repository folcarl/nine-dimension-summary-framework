# Quickstart

The default workflow starts from text or a transcript.

## Use The Main Skill

Ask Codex:

```text
Use $nine-dimension-summary to summarize this transcript in Chinese.
```

For English:

```text
Use $nine-dimension-summary to produce a full-mode English summary of this article.
```

For a short source:

```text
Use $nine-dimension-summary in lightweight mode for this short text.
```

## Typical Inputs

- pasted text;
- `.txt` transcript;
- article draft;
- interview transcript;
- lecture notes;
- research report notes;
- user-provided video-derived text.

## Default Output

The skill returns Markdown. For dense material it defaults to full mode. For short, low-density material it can use lightweight mode.

Media transcription is optional. If you already have text, do not use the transcription adapter.
