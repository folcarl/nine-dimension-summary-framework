# Scoring Rubric

Use a 10-point scale. Scores may end in `.0` or `.5` only. Do not use false precision such as 7.2.

## Anchors

| Score | Meaning |
| --- | --- |
| 9.0-10.0 | Exceptional: strong enough to serve as a benchmark for this type of material. |
| 8.0-8.5 | Very strong: substantial value with only local gaps. |
| 7.0-7.5 | Good: clear value, but visible limits in evidence, reasoning, or coverage. |
| 6.0-6.5 | Usable: provides information or insight, but must be handled cautiously. |
| 5.0-5.5 | Thin: offers limited clues, weak structure, or weak evidence. |
| Below 5.0 | Poor: confused, unsupported, misleading, or mostly unusable. |

If a dimension is below 6.0, name the main weakness. If a dimension is above 8.5, explain the exceptional strength.

## General Dimensions

| Dimension | What to evaluate |
| --- | --- |
| Information density | Specific facts, people, events, data, cases, scenes, or firsthand material. |
| Argument quality | Whether the reasoning chain connects problem, evidence, mechanism, and conclusion. |
| Evidence reliability | Source clarity, data口径, citation quality, verifiability, and separation of fact from inference. |
| Structural insight | Whether the material reveals deeper institutional, industrial, psychological, historical, or power structures. |
| Critical value | Whether it exposes blind spots, counterexamples, alternative positions, and viewing risks. |
| Practical judgment value | Whether it helps readers form judgment frameworks, observation lists, or decision boundaries. |
| Expression quality | Clarity, rhythm, restraint, and whether the argument avoids emotional shortcuts. |

## Specialized Dimensions

For investment/economic material, add when useful:

- Investment reference value
- Variable clarity
- Risk and counterexample coverage

For political/history material, add when useful:

- Historical context
- Position balance
- Ethical sensitivity

For interview/oral-history material, add when useful:

- Witness value
- Narrative restraint
- Fact verifiability

For art/film/documentary material, add when useful:

- Aesthetic completion
- Form/content relation
- Emotional aftereffect

Scoring should assess material quality, not whether the summarizer agrees with the source.
