# Completion Checklist

Run this checklist before finalizing.

## Mode

- [ ] Did I choose full or lightweight mode?
- [ ] If lightweight mode was used, is the material truly short, low-density, low-complexity, and low-stakes?
- [ ] If the material is long, dense, political, historical, economic, investment-related, research-based, or ambiguous, did I use full mode?

## Structure

- [ ] Did I restore the core content before evaluating it?
- [ ] Did I identify the real deeper problem in a standalone section or clearly visible passage?
- [ ] Did I discuss form, medium, audience, and viewing/reading risk?
- [ ] Did I reconstruct the evidence chain rather than only listing facts?
- [ ] Did I provide concrete critique: logic errors, evidence gaps, alternative positions, or framework risks?
- [ ] Did I separate source claims, inference, and unverified claims?
- [ ] Did I mark likely transcript errors, missing charts, unclear data口径, or facts needing external checks?
- [ ] Did I include practical implications without overclaiming?
- [ ] Did I include a scored evaluation table?

## Quality

- [ ] Is the summary readable as an essay rather than a rigid form?
- [ ] Are empty phrases replaced with concrete objects of concern?
- [ ] Are different positions specific rather than just attitude labels?
- [ ] Are economic or investment implications framed as judgment support rather than advice?
- [ ] Does the output language match the user request or source language?
